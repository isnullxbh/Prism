# Prism

C++ entity metadata extraction tool

![status](https://badgen.net/static/status/dev/yellow)
![status](https://badgen.net/static/c++/23/green)

## Prerequisites

- CMake
- C++ compiler with C++23 support

## Features

- Entities
  - [ ] Classes
  - [ ] Class templates
  - [ ] Enumerations
  - [ ] Functions
  - [ ] Function templates
  - [ ] Type aliases
  - [ ] Type alias templates
  - [ ] Attributes (annotated)
- Build systems
  - [ ] CMake
  - [ ] MSBuild
- Source types
  - [ ] Regular
  - [ ] Modules
